## Requirements:
* Node JS
* Metamask wallet
* Flask to run ML service

## Getting Started with Frontend

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.

### To Run the Flask service for ML model training,
